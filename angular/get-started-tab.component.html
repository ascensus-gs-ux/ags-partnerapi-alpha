Frequently asked questions

What do I need before I can call an API? ⌄

A generated feed API key, your egress IP ranges registered with Ascensus for non-production access, and a registered X.509 public certificate used to sign your client and user assertions.

How do I get my API key? ⌄

Open the Configure APIs tab, expand an application, and enable the feed you need.

Which functions use an API vs. an MFE? ⌄

Contributions are handled through a direct API integration. Withdrawals and Exchanges are delivered as Micro Front Ends (MFEs) that you embed and reach through Partner SSO.

Why was my contribution or update rejected? ⌄

The most common causes are account status and freeze codes. Only accounts in status 91 can be modified. The Troubleshoot tab lists each error.

When are APIs available? (Federal holiday schedule) ⌄

APIs accept requests 24/7. Transaction posting and settlement follow the U.S. financial calendar.
